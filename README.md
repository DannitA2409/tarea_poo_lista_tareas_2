# Lista de Tareas - POO

Proyecto en Python que implementa una lista de tareas con interfaz grafica usando `Tkinter`. La aplicacion permite agregar, marcar como completadas y eliminar tareas, aplicando una estructura basada en Programacion Orientada a Objetos.

## Funcionalidades

- Agregar nuevas tareas.
- Marcar tareas como completadas.
- Eliminar tareas existentes.
- Visualizar el estado de cada tarea en una tabla.

## Estructura del proyecto

- `main.py`: punto de entrada de la aplicacion.
- `modelos/tarea.py`: define la clase `Tarea`.
- `servicios/tarea_servicio.py`: gestiona la logica de negocio de las tareas.
- `ui/app_tkinter.py`: contiene la interfaz grafica.

## Requisitos

- Python 3
- Tkinter (incluido normalmente con Python)

## Ejecucion

```bash
python main.py
```
